<H1>CARA MEMBUAT PROJECT : </H1>
<ul>
  <li>
    Buat Folder baru<br>
  </li>
<li>
  masuk kedalam folder dengan perintah <b> cd nama_folder</b> dari terminal<br>
</li>
<li>
  kenalkan project dalam folder dengan perintah npm init membuat package.json ini sesuai dengan identitas project di akhiri dengan ketik"yes"<br>
</li>
<li>
  install depedenci/package express dengan perintah <b>"npm i express"</br>
</li>
<li>
  buat file <b>index.js</b> untuk file utama project "Server ada disini"<br>
</li>
<li>
  untuk menjalankan gunakan perintah <b>"node index.js"<br></b>
</li>
<li>
  7.Tunggu proses dan kemudian buka browser dan ketikkan http://localhost:port<br>
</li>
</ul>
